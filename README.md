# Online-Photo-Editor
Manipulate Your Image (By using filters) This is a mini project done by me.
Its basically a web application that applies filters to your uploaded photos. 

Filters available : 1. Blur Filter 
                    2. Make It Rainbow 
                    3. Red Filter 
                    4. Grayscale Filter 
                    5. Brighten It 
                    6. Contrast 
                    7. Negative Filter 
                    8. Saturation 
                    9. Shadow
